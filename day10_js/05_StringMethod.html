가장 많이 다루는 데이터 타입:String

  • 문자열 데이터는 index(0부터 시작)를 가지고 있음.
  • 문자열 값 내에서 개행(br)하는 것을 추천하지 않는다.
  • new String():객체 타입으로 선언 /추천하지않음
  • length:문자열의 길이
  • 문자관련 메서드:java에서 사용하는 String 메서드와 매우 비슷
  • 추출 관련 메서드: substring(),substr(),slice()
  • 문자 대체 메서드:replace(old,new):첫번째 매칭되는 문자만,대소문자 구분
  • trim():양쪽 공백 제거
  • padStart(길이,채울문자):앞쪽에서 해당길이만큼 문자를 채워준다. padEnd(길이,채울문자): 뒤쪽에서 해당길이만큼 문자를 채워준다. ex- 주민등록번호 뒷자리
  • charAt(index):해당 index 문자 추출
  • indexOf("str"):해당 문자열의 위치 반환
  • startWith("str")/endWith("str"): 해당 문자열이 앞/뒤에 존재하는지 확인하는 메서드 return타입은 true/false
